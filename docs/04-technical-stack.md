# Technical Stack

## Application

- Next.js.
- React.
- TypeScript.
- App Router.

## Styling

- Tailwind CSS.
- shadcn/ui.
- CSS variables.
- lucide-react.

## AI

- Official OpenAI JavaScript/TypeScript SDK.
- OpenAI Responses API.
- Server-side API route.
- Streaming responses.

## Validation

- Zod.

## Markdown

- react-markdown.
- remark-gfm.
- A syntax-highlighting library compatible with the application architecture.

## State Management

For the MVP:

- React state.
- Custom hooks.
- AbortController.

Do not add a state-management library until it is clearly required.

## Database

The first MVP may not use a database.

A later version may use:

- PostgreSQL.
- Prisma or Drizzle ORM.

## Authentication

A later version may use:

- Auth.js.
- Another authentication provider selected through an architecture decision.

## Deployment

- Vercel.
- Any compatible Node.js hosting platform.
